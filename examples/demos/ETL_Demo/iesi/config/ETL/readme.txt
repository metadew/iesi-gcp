This file can be opened with Pentaho Data Integration. (Created with PDI 8.2)

It expects these parameters when executed:

ordersFile : Path to the orders.csv
productsFile : Path to the products.csv
targetFile : Path to the target excel file
dbHost : hostname of the target PG database
dbPort : port of the target PG database
dbUser : user of the target PG database, this will also be used for the databasename
dbPass : password of the target PG database
